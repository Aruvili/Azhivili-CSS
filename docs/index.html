Features

Tamil Class Names

All utility classes use Tamil names for better localization and cultural relevance.

Production Ready

Optimized SCSS architecture with proper organization and build tools.

Responsive Design

Mobile-first responsive utilities with Tamil breakpoint names.

Modern CSS

Uses latest CSS features including Grid, Flexbox, and custom properties.

Comprehensive

Complete set of utilities for spacing, typography, colors, and layout.

Customizable

Easy to customize and extend with SCSS variables and mixins.
